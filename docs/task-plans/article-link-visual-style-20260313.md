# 文章链接视觉样式修复方案

## 问题确认

- 文章详情页正文中的 URL 已经被渲染为 `<a>`。
- 但链接没有使用显式的链接色和下划线，视觉上仍接近普通正文。
- QA 区回答文本带有 `text-muted-foreground`，其中的链接也会继承成灰色。

## 修复范围

控制在 3 个文件内：

1. `src/app/[lang]/articles/[slug]/page.tsx`
2. 1 个测试文件
3. 本方案文档

## 修复思路

1. 为正文 HTML 容器增加显式链接样式：
   - 默认使用 `text-primary`
   - 悬停时加深
   - 保持下划线和合理的断行
2. 为 QA 回答文本增加同样的链接样式，避免继承成灰色。
3. 不引入 typography plugin，本次只做局部样式收口，避免扩大范围。

## 验收标准

- 文章正文中的链接在 HTML 视图下显示为明显的链接色。
- QA 回答中的链接也显示为明显的链接色。
- 现有链接跳转能力不变。
- 页面其余文本颜色不被误伤。
