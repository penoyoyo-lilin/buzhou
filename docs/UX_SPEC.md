# UX 规范文档 (UX Specification)

## 设计原则

### 核心原则
1. **AI-First Design**：优先考虑 AI Agent 的信息获取效率
2. **双模一致性**：人类视图与 Agent 视图信息同源、逻辑一致
3. **信息密度优先**：减少装饰性元素，最大化有效信息
4. **渐进式引导**：从 Web 浏览到 API 调用的自然过渡

## 视觉风格

### 色彩系统

| 用途 | 色值 | 使用场景 |
|------|------|----------|
| **主色** | `#2563EB` (Blue-600) | 主要按钮、链接、品牌元素 |
| **成功** | `#10B981` (Green-500) | 验证通过、成功状态 |
| **警告** | `#F59E0B` (Amber-500) | 部分验证、需注意 |
| **错误** | `#EF4444` (Red-500) | 验证失败、错误状态 |
| **中性** | `#6B7280` (Gray-500) | 次要文本、边框 |
| **背景** | `#F9FAFB` (Gray-50) | 页面背景 |

### 排版规范

| 元素 | 字体 | 字号 | 字重 |
|------|------|------|------|
| **标题 H1** | Inter / 思源黑体 | 32px | 700 |
| **标题 H2** | Inter / 思源黑体 | 24px | 600 |
| **标题 H3** | Inter / 思源黑体 | 20px | 600 |
| **正文** | Inter / 思源黑体 | 16px | 400 |
| **代码** | JetBrains Mono | 14px | 400 |
| **标签** | Inter / 思源黑体 | 12px | 500 |

### 间距系统
- **基础单位**：4px
- **常用间距**：8px, 12px, 16px, 24px, 32px, 48px

## 组件规范

### 文章卡片

```
┌─────────────────────────────────────────────────────┐
│ [验证状态徽章]                        [置信分数]    │
│                                                     │
│ 文章标题 - 核心问题简述                              │
│                                                     │
│ [标签1] [标签2] [标签3]              更新于 X 天前   │
└─────────────────────────────────────────────────────┘
```

**交互规则**：
- 悬停：边框高亮 + 微微上浮
- 点击：跳转详情页
- 验证状态颜色编码：通过(绿)、部分(黄)、失败(红)

### 验证状态徽章

| 状态 | 样式 | 文案 |
|------|------|------|
| 已验证 | 绿色背景 + 勾选图标 | "已验证" |
| 部分验证 | 黄色背景 + 警告图标 | "部分验证" |
| 未验证 | 灰色背景 + 问号图标 | "待验证" |
| 已失效 | 红色背景 + 叉号图标 | "已失效" |

### 详情页布局

### 视图切换

**人类视图**：
- 优化的排版和间距
- 代码高亮
- 图片懒加载
- 平滑滚动

**Agent 视图**：
- 纯 Markdown / JSON 输出
- 无装饰性元素
- 全量元数据
- 代码块原始文本

## 响应式断点

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| **Mobile** | < 640px | 单列布局，隐藏次要信息 |
| **Tablet** | 640px - 1024px | 双列布局，侧边栏折叠 |
| **Desktop** | > 1024px | 完整三列布局 |

## 国际化规范

### 语言切换
- 顶部导航显眼位置
- 当前语言高亮
- 切换后保持当前页面路径

### 内容规则
- UI 文本：必须翻译
- 用户生成内容：保持原文，标注语言
- API 响应：根据 Accept-Language 返回

---

## 可访问性要求

### WCAG 2.1 AA 标准
- 颜色对比度 >= 4.5:1
- 键盘导航支持
- 屏幕阅读器兼容
- 焦点状态可见

### AI 友好设计
- 关键数据同步渲染（非异步 JS）
- HTML 注释内嵌 API 引导指令
- Meta 标签声明 API 入口
- 结构化数据标记（Schema.org）
