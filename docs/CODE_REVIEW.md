## 2. 安全问题详情

### 2.1 登录失败锁定 (中风险)

**问题：** 当前登录失败后没有锁定机制，可能被暴力破解。

**修复建议：**
```typescript
// 添加登录失败计数
// 5 次失败后锁定 15 分钟
```

### 2.2 Session 刷新 (低风险)

**问题：** Session 创建后 2 小时过期，用户活跃时不会延长。

**修复建议：**
```typescript
// 在 auth/me 中检查并刷新 Session
if (session.expiresAt > new Date(Date.now() + 30 * 60 * 1000)) {
  // 距离过期还有 30 分钟以上，不刷新
} else {
  // 刷新 Session
  await prisma.session.update({
    where: { id: session.id },
    data: { expiresAt: new Date(Date.now() + 2 * 60 * 60 * 1000) }
  })
}
```
