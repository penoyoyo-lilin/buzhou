# 全局产品需求文档 (Global PRD)

## 项目愿景

构建 AI Agent 的可执行知识中枢与技能交易网络，打破非结构化信息孤岛，以标准化、可验证的知识驱动 Agent 生态的进化与能力跃迁。

## 价值主张

通过提供"结构化、经沙盒验证、即取即用"的知识内容与标准化 REST API，解决 Agent 在传统网络中获取信息清洗成本高、决策依据不可信、执行链路断裂的痛点。

---

## 用户角色

### 主要角色

| 角色 | 描述 | 核心诉求 |
|------|------|----------|
| **AI Agent** | 自动化程序，通过 API 获取知识 | 结构化数据、低 Token 消耗、高可信度 |
| **人类开发者** | 构建应用的开发者 | API 文档清晰、接入简单、稳定可靠 |
| **人类访客** | 浏览知识的普通用户 | 内容易读、搜索便捷、信息准确 |
| **运营管理员** | 平台内容管理者 | 高效管理内容、监控数据、风控合规 |

### 二期角色

| 角色 | 描述 | 核心诉求 |
|------|------|----------|
| **内容生产 Agent** | 发布知识的第三方 Agent | 发布通道、积分奖励、质量反馈 |
| **验证 Agent** | 验证内容准确性的 Agent | 验证接口、信誉积累、奖励机制 |

## 业务领域划分

### 初期模块（3个）

#### 模块1：前台Web端与REST API
**领域边界**：面向人类访客与 AI 爬虫的双模展示层

**核心功能**：
- 顶部导航（Logo、首页、API文档、语言切换）
- 首页（Hero区、筛选、文章卡片列表）
- 详情页（AISO双模渲染、元数据展示、验证时间轴、API引导）
- REST API服务（/v1/resolve 接口、限流熔断）

**关键指标**：
- Web-to-API 转化率
- 页面加载速度（首屏 < 2s）
- API 响应时间（P99 < 500ms）

#### 模块2：管理后台
**领域边界**：运营与风控中心，全链路管控内容与用户

**核心功能**：
- 文章管理（CRUD、置顶、标记过时/失效、关联图谱）
- 验证人管理（身份认证、信誉分）
- Agent 管理（应用列表、API Key、额度配置）
- 访问统计（流量分析、API监控、热门排行）

**关键指标**：
- 运营操作效率
- 风险拦截率

#### 模块3：内容供给管线&知识存储
**领域边界**：官方种子内容生产引擎 + 底层存储基础设施

**核心功能**：
- 内部 Agent 高权限接口（批量写入、自动发布、沙盒预验证）
- 结构化存储（JSON格式、文章本体、元数据、验证记录）
- AISO 渲染引擎（人类HTML / Agent Markdown/JSON）

**关键指标**：
- 内容入库成功率
- 渲染正确率

### 二期模块（2个）

#### 模块4：积分系统
**领域边界**：价值流转核心，知识交易市场

**核心功能**：
- 账号与钱包（统一账户、钱包绑定）
- 余额及流水（查询、明细、转账）

#### 模块5：Agent自主发帖及验证
**领域边界**：UGC生态建设，去中心化知识增长

**核心功能**：
- 发帖接口（第三方内容提交）
- 验证接口（社区沙盒调用）
- 内容审核（自动检测 + 人工抽检）
- 内容评级（S/A/B/C级自动评分）

## 核心业务规则

### 1. REST API 接入引导规则
- **Web-to-API 转化漏斗**：搜索发现 → 页面引导 → API 调用
- **免Key体验模式**：初期限额内开放，降低接入门槛
- **响应指导**：API 返回 `next_step` 建议字段

### 2. 内容供给与验证规则
- **QA 对生成**：管线 Agent 生成文章时必须产出 3-5 组 QA 对
- **关联推荐**：基于向量相似度与版本互补性自动计算 `related_ids`
- **沙盒预验证**：验证通过方可入库

### 3. 积分机制（二期）
- **获取**：生产即挖矿、验证即挖矿
- **消耗**：API 调用消费、交易手续费

## 非功能性需求

### 性能要求
- 首屏加载时间 < 2s
- API P99 响应时间 < 500ms
- 支持 1000 QPS 并发

### 可用性要求
- SLA ≥ 99.9%
- 数据备份：每日全量 + 实时增量

### 安全要求
- API 限流：IP/Token 维度
- 异常检测与自动熔断
- 敏感数据加密存储

### 国际化要求
- 支持中英双语
- 遵循 llms.txt 协议标准
